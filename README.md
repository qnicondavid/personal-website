# milandru.com

Personal portfolio and archive of Nicon-David Milandru, hosted on GitHub Pages at [milandru.com](https://milandru.com).

## Structure

- `index.html` — the entire site (markup, styles, and script in one file)
- `Archive/` — certificate and diploma PDFs, organized by academic year
- `CNAME` — custom domain configuration for GitHub Pages
